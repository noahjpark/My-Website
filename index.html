Software Development

Downloads ReRouter

As a teaching assistant, I took the initiative to stop plagiarism and protect the integrity of the University of Minnesota. To do this, I utilized the MOSS open source system publicly available from Stanford University. To generate a report of all relevant information on plagiarism from student submissions, I had to provide the MOSS script with the directory of student submissions. Creating this directory manually was largely time consuming, so I decided to write a script to automate the process of creating and populating the submission directory. The source code can be found on my GitHub.

Powerlifting Converter

Powerlifting Converter is a simple kilogram to pound calculator that provides the user with easy access to visually accurate kilogram plates to add to the current total. The app supports adding/removing various amounts of kilograms in the form of approved plates. Neither kilograms nor pounds can ever be a negative number. The GitHub repository linked below contains the source code for the application and a direct link to the Google Play Store where the application is live.
